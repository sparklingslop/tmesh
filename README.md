# tmesh

**tmux-native agent mesh. The real one this time.**

This is the proper mesh -- the one with topology, multi-hop routing, and the kind of architecture diagrams that make people nod thoughtfully in meetings.

It is being built. It will be very impressive, assuming it doesn't also fail into something better.

In the meantime, the thing that actually works right now is [tresh](https://github.com/sparklingslop/tresh) -- the Trivially Reliable Event Shuttle for Harnesses. It is not a mesh. It has made its peace with that.
